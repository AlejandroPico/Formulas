# Significado

La energía libre de Gibbs permite decidir si un proceso puede ser espontáneo cuando la temperatura y la presión se mantienen constantes.

`Delta H` mide el cambio de entalpía: si el sistema libera o absorbe calor a presión constante. `Delta S` mide el cambio de entropía: si aumenta o disminuye el número de configuraciones accesibles. La temperatura pondera la importancia del término entrópico.

Si `Delta G` es negativa, el proceso es termodinámicamente favorable. Si es positiva, no es espontáneo en esas condiciones. Si es cero, el sistema está en equilibrio.

El simulador usa una balanza visual: un lado representa la tendencia espontánea y el otro la no espontánea. También muestra el umbral térmico donde `Delta G` cambia de signo cuando `Delta S` no es cero.
