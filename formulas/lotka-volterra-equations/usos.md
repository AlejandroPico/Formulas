# Usos

- Modelos depredador-presa.
- Introducción a sistemas dinámicos no lineales.
- Estudio de oscilaciones ecológicas.
- Simulación de poblaciones acopladas.
- Enseñanza de ecuaciones diferenciales.
- Análisis cualitativo de estabilidad.

El modelo básico no incorpora capacidad de carga, migración, estacionalidad ni estructura espacial, pero captura una interacción mínima muy expresiva.
