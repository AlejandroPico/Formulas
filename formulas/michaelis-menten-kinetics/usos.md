# Usos

- Estimar `Vmax` y `Km` a partir de datos experimentales.
- Comparar actividad de enzimas.
- Analizar saturación por sustrato.
- Estudiar inhibición enzimática como extensión del modelo.
- Bioquímica metabólica y farmacología.
- Enseñanza de cinética de reacciones catalizadas.

El modelo describe condiciones iniciales y sistemas sencillos. No cubre por sí solo cooperatividad, regulación alostérica ni mecanismos complejos de múltiples sustratos.
