# Ficha

## Identificación

- Nombre: Curvatura de una curva.
- Área: geometría diferencial.
- Fórmula: `κ=|f''(x)|/[1+(f'(x))²]^(3/2)`.

## Variables

- `κ`: curvatura.
- `R`: radio de curvatura.
- `f'(x)`: pendiente local.
- `f''(x)`: cambio de pendiente.

## Lectura del simulador

La curva gris es una parábola. El punto rojo es el punto evaluado y el círculo azul es el círculo osculador. Un círculo pequeño indica mayor curvatura; un círculo grande indica menor curvatura.
