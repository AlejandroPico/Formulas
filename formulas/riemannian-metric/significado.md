# Significado

Una metrica riemanniana indica como medir longitudes, angulos y areas de forma local.

En coordenadas, el tensor metrico asigna una distancia a cada desplazamiento infinitesimal. Si sus componentes cambian, la geometria cambia aunque las coordenadas parezcan iguales.

El simulador muestra una malla deformada por el factor g11 y calcula el intervalo local ds.
