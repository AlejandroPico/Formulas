# Derivación

Una interpretación simple parte de una distribución de energías moleculares. Solo las moléculas cuya energía supera la barrera de activación contribuyen eficazmente a la reacción.

La fracción de moléculas con energía suficiente aparece de forma aproximada como un factor exponencial `exp(-E_a/(RT))`.

El factor `A` multiplica esa fracción y representa la frecuencia efectiva de intentos de reacción, incluyendo colisiones y orientación molecular.

Tomando logaritmos se obtiene una forma lineal: `ln k = ln A - E_a/R · 1/T`, útil para estimar `E_a` a partir de datos experimentales.
